sparkjs
=======

## Setup your dev environment

* Install your local dependencies:

```shell
$ npm install
```

* Install globally mocha, istanbul and jshint

```shell
$ npm install -g mocha
$ npm install -g istanbul
$ npm install -g jshint
```

## How to test

`make test`

### Lint your code

`make lint`

### Coverage report

`make cover`
